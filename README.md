# 🐍 Snake Game (Python + Pygame)

## 📌 Overview

This project is a classic Snake Game built using Python and Pygame.
It demonstrates core programming concepts like game loops, collision detection, object-oriented design, and event handling.

The game is designed with a clean UI and structured code, making it easy to understand and explain in interviews.

---

## 🎯 Features

* Classic snake movement using arrow keys
* Food generation at random positions
* Snake grows on eating food
* Collision detection (wall + self)
* Score tracking
* Increasing difficulty (speed increases with score)
* Start screen, pause functionality, and game over screen
* Restart and quit options

---

## 🧠 Concepts Used

* Object-Oriented Programming (OOP)
* Game Loop Design
* Event Handling (Keyboard inputs)
* Collision Detection
* Grid-based movement system

---

## 🛠️ Tech Stack

* Python
* Pygame

---

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/snake-game-python.git
cd snake-game-python
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Run the game:

```bash
python snake_game.py
```

---

## 🎮 Controls

| Key        | Action              |
| ---------- | ------------------- |
| Arrow Keys | Move snake          |
| SPACE      | Start game          |
| P          | Pause / Resume      |
| R          | Restart (Game Over) |
| Q          | Quit                |
